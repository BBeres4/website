My Projects

Here are some of the projects I’ve worked on:

OpenWeatherMap API

This website is a simple weather app that uses the OpenWeatherMap API to let users enter a city, state, and country and view the current conditions and a 5-day forecast in Fahrenheit.

DCF Model

This model estimates a company’s intrinsic value by forecasting future cash flows and discounting them back to today to show what the business is truly worth. (Excel file).

LBO Model

This model analyzes a leveraged buyout by projecting cash flows, debt repayment, and investor returns to show how profitable a debt-funded acquisition could be. (Excel File)

Portfolio Tracker

This project is a simple web-based portfolio tracker that lets users input stock holdings and view real-time values, gains, and overall performance.

Personal Budget App

This app helps users manage monthly expenses by categorizing spending, tracking budgets, and visualizing financial habits through charts.

SQL Sample Database

This project includes a sample SQL database and queries demonstrating joins, aggregations, and data analysis for practice and learning.
